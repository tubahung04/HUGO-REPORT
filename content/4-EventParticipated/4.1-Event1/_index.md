### Event Information
* **Event Name**: AWS Cloud & Data Pipeline Workshop
* **Time**: May 2026
* **Location**: HCMC / Online
* **Role**: Attendee

### Main Content
The event focused on sharing the latest Serverless architecture trends on AWS, especially how to build cost-optimized Data Pipelines using Kinesis, S3, and Athena. Speakers presented real-world case studies on handling millions of Logs per minute.

### Lessons Learned & Personal Contribution
The knowledge from this event was the biggest inspiration for me to design my **Hybrid SOC Dashboard** system. I learned not to use traditional SQL databases for Logs, but rather an S3 Data Lake. This knowledge was directly applied to my internship report project!
